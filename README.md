# 🔐 CyberScan – Web Penetration Testing Tool

## Project Description
CyberScan is a semi-automated web-based penetration testing tool built using Flask. It allows users to input a target URL and performs vulnerability scanning using Nikto.


## Features
- Web-based dashboard
- Scan any website URL
- Integration with Nikto tool
- Displays vulnerability results
- Generates scan report


## Technologies Used
- Python (Flask)
- HTML, CSS
- Nikto (Security Scanner)


##  How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
python app.py

3. Open browser:
http://127.0.0.1:5000
